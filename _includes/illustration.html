{% comment %} Illustration include This include adds a figure to the page styled using bootstrap, https://getbootstrap.com/docs/5.1/content/figures/ It requires an "objectid" with the include, which is used to find the object title and image (for collection items). If multiple collection objectids are included (separated by ; ), they will be added to a "col-md" in a row which will automatically divided equally. Alternatively, a URL to an external image can be used in "objectid". E.G. --> {% include illustration.html img="demo_001.png" %} Options: - "img" = filename of item in "illustrations/" folder - "alt" = alternative text describing the image. This is a required accessibility feature - "caption-top", "caption-bottom", "caption-left", "caption-right" - "class" {%- endcomment -%}
